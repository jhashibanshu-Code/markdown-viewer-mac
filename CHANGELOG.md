# Changelog

## 0.1.0

- Added secure Markdown editing and preview surfaces.
- Added direct file opening, vault opening, local save, save-as, autosave recovery, and version history foundations.
- Added graph and mind-map views for Markdown vault navigation.
- Added local static publishing with search data, backlinks, and graph JSON.
- Added CLI commands for context export, static publish, note creation, and app URL generation.
- Added Claude/ChatGPT context export files, including navigation guide, mind map, and graph JSON.
- Added a repo-packaged LLM skill for using the viewer as a local context system.
- Added macOS packaging, generated icons, ZIP artifact generation, checksums, download page generation, and release readiness checks.

Known release status: development builds are ad-hoc signed. Public macOS distribution still requires Developer ID signing, notarization, stapling, and a manual desktop launch smoke test.
