# Third Party Notices

Shibanshu Markdown Viewer uses open source packages from the JavaScript and Electron ecosystem. Dependency versions are recorded in `package-lock.json`.

Primary runtime dependencies include:

- Electron
- Vite
- CodeMirror
- DOMPurify
- Marked
- Mermaid (includes D3, Cytoscape, dagre-d3, and other sub-dependencies)
- KaTeX
- highlight.js
- Lucide Icons
- github-markdown-css

Each dependency remains under its own license. Before a public release, regenerate this notice from the exact packaged dependency graph and include the full license text required by each dependency.

Current readiness status: this notice is present as trust material, but full legal review and license aggregation should be completed before broad public distribution.
